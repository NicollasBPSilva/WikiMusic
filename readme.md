# Projeto WikiMusic

### Requitos

- Backend
- Frontend
- Banco de dados

### Funcionalidades

- Exibe infos. de albuns musicais;
- Exibe infos. dos artistas;
- Exibe a música mais tocada do artista;
- Álbuns relacionados;
- Álbum mais tocado no ano;
- Música com mais stream;
- Músicas premiadas;
